This is just the second blog post. The inspiration for doing this came from this article. Other options for static website generators are Jekyll written in Ruby, Hyde written in Python, Hexo written in Javascript (nodeJS), Hugo written in Go, Cactus written in Python and using Django, Lektor in Python and using Jinja2, Sphinx written in Python and using rST as markup language, Nikola written in Python it uses rST as default markup, but supports many more. It also has a plugin for Mediawiki markup. Lots more on StaticGen.

Hmm, it looks like Lektor also has support for Github pages.
