Github cheatsheets

Tue 27 February 2018
By Torfinn Ingolfsen
Category: Blog

There are two slightly different git "cheat sheets" that you can download from Github, one blue and one green in color. The blue one refers to Github Training while the green one refers to Github Education. Also, conetents and wording is slightly different between them. Still, they are both useful.
